# START HERE

이 사이트는 **문서 생산용**이다.

## 특징

- PWA 없음 (의도적)
- 브라우저 번역 사용 가능
- push = 자동 배포

## 작업 방법

1. `src/content/{카테고리}/{slug}.mdx` 생성
2. `git push origin main`
3. 끝

## 금지 사항

- 루트에 MD 파일 추가 금지
- PWA/ServiceWorker 코드 추가 금지
- manifest 파일 추가 금지

> 위반 시 빌드 자동 실패 (repo-guard.mjs)
