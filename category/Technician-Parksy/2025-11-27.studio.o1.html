0. System Overview

PARKSY ROOM OS is a creative operating system built inside a single rectangular room in Anyang, Korea. The room is not treated as “interior decoration” but as a physical kernel that maps four absolute directions – North, East, South, West – to four cognitive modes: Reset, Production, Relaxation, Meta-Structure.

1. North (0°) – Archive & Reset Node

Recliner · Bookshelf · Wall TV (Threads Canvas)

NORTH PHOTO FRAME

1.1 Functional Role

Facing straight North, the wall hosts a TV used as a thread canvas. This screen cycles through PARKSY’s active projects, GitHub pages, and long-term “season logs”. Beneath and around it sit the core bookshelf and working bags.

  • Archive & Memory Core – finished seasons, reference books, printed diagrams.
  • Low-arousal Reset – recliner + TV for passive review.
  • Anchor for Absolute North – all other directions are measured from this wall.

1.2 Cognitive Mapping

Hippocampal archive state. Default Mode Network softly on. Place & Grid cells define “home base”.

1.3 OS Contract

No new fights started here. Only review, passive scroll, mood reset.

1.4 Feng Shui

North = water, depth, memory. Recliner = stable spine. Bookshelf = mountain behind the house.
